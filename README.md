# 🗂️ Event Task Manager Agent
Built with **Google ADK + Groq** · MongoDB · GDG Cloud Chennai 🎉

## Secrets to add in HF Spaces
| Key | Value |
|---|---|
| `GROQ_API_KEY` | From [console.groq.com](https://console.groq.com) |
| `MONGODB_URI` | From MongoDB Atlas free cluster connection string |

## Features
- 📝 Create & track event tasks
- 👥 Log speakers & connections
- 📚 Save key learnings
- 🔁 Restore previous sessions via MongoDB
